
# ATOM Lite Pinout & Specs

## Pinout

| Pin      | Function    |
|----------|-------------|
| 3V3      | Power       |
| G22      |             |
| G19      | MOSI        |
| G23      | CLK         |
| G33      | MISO / ADC  |
| G21      | SCL         |
| G25      | DAC / SDA   |
| 5V (x2)  | Power       |
| GND (x2) | Ground      |
| G26      | DAC         |
| G32      | ADC         |
| G27      | RGB LED     |
| G37      | Button      |
| G12      | IR          |

## Size

**Dimensions:** 81.0 × 65.0 × 13.0 mm
